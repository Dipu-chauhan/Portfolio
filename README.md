# Modern Portfolio Website

A beautiful and responsive portfolio website built with Flask and modern CSS.

## Features

- Responsive design that works on all devices
- Modern and clean user interface
- Smooth scrolling and animations
- Contact form
- Project showcase section
- Skills display
- Social media links

## Setup Instructions

1. Clone this repository
2. Create a virtual environment:
   ```
   python -m venv venv
   ```
3. Activate the virtual environment:
   - Windows:
     ```
     venv\Scripts\activate
     ```
   - Mac/Linux:
     ```
     source venv/bin/activate
     ```
4. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
5. Run the application:
   ```
   python portfolio.py
   ```
6. Open your browser and navigate to `http://localhost:5000`

## Customization

1. Edit the content in `templates/index.html` to add your personal information
2. Modify the styles in `static/css/style.css` to change the appearance
3. Add your project images to `static/images/` and update the image paths in the HTML
4. Update the contact information and social media links

## Technologies Used

- Flask
- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (Inter)
